命令名称 GetMemoryDataBmp 获取图像数据内存地址
命令功能 获取图像数据的内存地址和数据长度
语法格式 TURING.GetMemoryDataBmp(data, size)
参数说明 data：整数型，返回图像数据内存地址
size：整数型，返回图像数据字节长度
返回值 无
按键精灵
复制代码 
'获取屏幕像素数据
Call TURING.Pixel_FromScreen(0, 0, 300, 180)
'返回图像数据内存地址
Call TURING.GetMemoryDataBmp(data, size)
'获取内存地址图像数据
Call TURING.Pixel_FromMemory(data, size)
'显示获取的图像
Call TURING.Pixel_Preview()
