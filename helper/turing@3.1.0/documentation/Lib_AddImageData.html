命令名称 Lib_AddImageData 图像数据追加为字库
命令功能 内部的图像数据追加为识别字库
语法格式 TURING.Lib_AddImageData(text)
参数说明 text：字符串型，字符点阵对应的识别文字
返回值 无
按键精灵
复制代码 
'创建字库（准备一张10个数字的图片作为字库）
Call TURING.Pixel_FromPicture("C:\10个数字.bmp")
'二值化处理
Call TURING.Filter_Binaryzation("128-255")
'显示图像预览
Call TURING.Pixel_Preview()
'固定位置切割
数量 = TURING.Incise_FixedLocation(6, 4, 7, 11, 13, 10)
Dim n(9)
文字 = "0123456789"        
For i = 0 To 9
    n(i) = mid(文字, i + 1, 1)  '设置字库对应的文字
Next
For i = 0 To 数量
    '载入切割的字符图像数据
    Call TURING.Pixel_SetImageDataCut(i) 
    '设置点阵识别库
    Call TURING.Lib_AddImageData(n(i))   
Next
'识别测试
