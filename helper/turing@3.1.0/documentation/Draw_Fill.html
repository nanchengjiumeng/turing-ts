命令名称 Draw_Fill 绘图填充
命令功能 在内部的图像数据上对指定坐标开始进行颜色填充
语法格式 Area = TURING.Draw_Fill(x, y, Through, R, G, B)
参数说明 x：整数型，开始X坐标
y：整数型，开始Y坐标
Through：布尔型，可选，是否八向,默认False
R：整数型，可选，默认255，10进制红色分量值
G：整数型，可选，默认0，10进制绿色分量值
B：整数型，可选，默认0，10进制蓝色分量值
返回值 Area：字符串型，返回填充区域范围（格式：左,上,右,下）
按键精灵
复制代码 
'创建一个空白画布
Call TURING.Draw_CreateCanvas(100, 100)
'画圆
Call TURING.Draw_Circle(50, 50, 25, 255, 255, 255)
'显示图像
Call TURING.Pixel_Preview()
'颜色填充
TracePrint TURING.Draw_Fill(1, 1, false, 255, 255, 255)
TracePrint TURING.Draw_Fill(50, 50)
'显示图像
Call TURING.Pixel_Preview()
