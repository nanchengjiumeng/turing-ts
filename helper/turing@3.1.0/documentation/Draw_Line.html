命令名称 Draw_Line 绘图画线
命令功能 在内部的图像数据上画一条直线
语法格式 TURING.Draw_Line(x1, y1, x2, y2, R, G, B)
参数说明 x1：整数型，起点X坐标
y1：整数型，起点Y坐标
x2：整数型，终点X坐标
y2：整数型，终点Y坐标
R：整数型，可选，默认255，10进制红色分量值
G：整数型，可选，默认0，10进制绿色分量值
B：整数型，可选，默认0，10进制蓝色分量值
返回值 无
按键精灵
复制代码 
'获取屏幕区域像素
Call TURING.Pixel_FromScreen(0, 0, 100, 100)
'画线
Call TURING.Draw_Line(0, 20, 100, 20, 255, 0, 0)
'显示图像
Call TURING.Pixel_Preview()
