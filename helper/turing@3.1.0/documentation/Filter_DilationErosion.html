命令名称 Filter_DilationErosion 腐蚀膨胀
命令功能 (针对黑白图像处理)腐蚀膨胀滤镜
语法格式 TURING.Filter_DilationErosion(Model)
参数说明 Model：整数型，模式（默认：1膨胀，0腐蚀）
返回值 无
按键精灵
复制代码 
'获取屏幕图像数据
Call TURING.Pixel_FromScreen(0,0,200,100)
'二值化处理
Call TURING.Filter_Binaryzation("128-255")
'显示获取的图像
Call TURING.Pixel_Preview()
'对白色字符进行膨胀
Call TURING.Filter_DilationErosion()
'图像预览
Call TURING.Pixel_Preview()
滤镜效果
