命令名称 Draw_Ellipse 绘图画椭圆
命令功能 在内部的图像数据上画一个椭圆形
语法格式 TURING.Draw_Ellipse(x, y, wRadius, hRadius, R, G, B, Full, Alpha)
参数说明 x：整数型，圆心X坐标
y：整数型，圆心Y坐标
wRadius：整数型，椭圆的宽半径
hRadius：整数型，椭圆的高半径
R：整数型，可选，默认255，10进制红色分量值
G：整数型，可选，默认0，10进制绿色分量值
B：整数型，可选，默认0，10进制蓝色分量值
Full:布尔型，可选，默认False，填充实心为True
Alpha:整数型，可选，透明度（默认100不透明，需要填充实心才能生效）
返回值 无
按键精灵
复制代码 
'获取屏幕区域像素
Call TURING.Pixel_FromScreen(0, 0, 150, 150)
'画椭圆
Call TURING.Draw_Ellipse(30, 30, 15, 25, 255, 0, 0)'空心
Call TURING.Draw_Ellipse(50, 50, 15, 25, 255, 0, 0, True, 50)'实心半透明
Call TURING.Draw_Ellipse(70, 70, 15, 25, 255, 0, 0, True)'实心
'显示图像效果
Call TURING.Pixel_Preview()
