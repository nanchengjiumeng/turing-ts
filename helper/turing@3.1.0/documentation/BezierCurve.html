命令名称 BezierCurve 贝塞尔曲线
命令功能 生成贝塞尔曲线轨迹数据
语法格式 Pos = TURING.BezierCurve(coordinates,stepping)
参数说明 coordinates：字符串型，坐标节点(格式："x坐标,y坐标|x坐标,y坐标|…")
stepping：小数型，步进
返回值 Pos：字符串型，曲线上所有坐标（格式：x坐标,y坐标-x坐标,y坐标…）
按键精灵
复制代码 
Dim 节点(3)
节点(0) = "30,60"
节点(1) = "250,30"
节点(2) = "550,30"
节点(3) = "600,60"
步进 = 0.02
'得到贝塞尔曲线轨迹
数据 = TURING.BezierCurve(Join(节点, "|"), 步进)
'创建画面（演示）
Call TURING.Draw_CreateCanvas(650, 100)
轨迹 = Split(数据, "-")
For i = 0 To UBound(轨迹)
    坐标 = Split(轨迹(i), ",")
    x = Clng(坐标(0))
    y = Clng(坐标(1))
    '绘制坐标（演示）
    Call TURING.Draw_Circle(x, y, 3, 255, 0, 0, True, 50)
Next
'图像显示（演示）
Call TURING.Pixel_Preview()
