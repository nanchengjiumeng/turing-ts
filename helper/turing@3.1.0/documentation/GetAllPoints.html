命令名称 GetAllPoints 获取所有端点坐标
命令功能 获取内部图像数据上的所有认为是端点的坐标
语法格式 Pos = TURING.GetAllPoints(Value)
参数说明 Value：整数型，可选，阈值（默认7，1~上限8）
返回值 Pos：字符串型，所有端点的坐标（格式:x坐标,y坐标|…）
按键精灵
复制代码 
'获取文件图像数据
Call TURING.Pixel_FromPicture("C:\验证码.BMP")
'二值化处理
Call TURING.Filter_Binaryzation("0-128")
'提取所有端点坐标
数据 = TURING.GetAllPoints(7)
端点 = Split(数据, "|")
数量 = UBound(端点)
For n = 0 To 数量
    TracePrint 端点(n)   '输出坐标数据
    坐标 = Split(端点(n), ",")
    x = 坐标(0)
    y = 坐标(1)
    'TracePrint x
    'TracePrint y
    '画颜色点
    Call TURING.Draw_Point(x, y)
Next
'显示端点图像
Call TURING.Pixel_Preview()
