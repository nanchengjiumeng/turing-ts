命令名称 Draw_Recover 图像数据恢复
命令功能 【覆盖】恢复内部的备份图像数据为其他模式的数据
语法格式 TURING.Draw_Recover(Serial, Mode)
参数说明 Serial：整数型，备份编号（范围：1~4096）
Mode：整数型，模式（默认0：内部图像数据，1：图中图数据，2：绘图数据）
返回值 无
按键精灵
复制代码 
'获取屏幕区域像素
Call TURING.Pixel_FromScreen(0, 0, 100, 100)
Call TURING.Draw_Point(50, 60, 255, 0, 0)        '画点
'进行图像数据备份（编号：1~4096）
Call TURING.Draw_Backups(1)
Call TURING.Draw_Line(0, 20, 100, 20, 255, 0, 0) '画线
'进行图像数据备份
Call TURING.Draw_Backups(2)
Call TURING.Draw_Circle(50, 60, 25, 255, 0, 0)   '画圆
'进行图像数据备份
Call TURING.Draw_Backups(3)
'恢复备份的图像数据（编号：1~4096）
Call TURING.Draw_Recover(3)
'显示图像
Call TURING.Pixel_Preview()
'恢复备份的图像数据
Call TURING.Draw_Recover(2)
'显示图像
Call TURING.Pixel_Preview()
'恢复备份的图像数据
Call TURING.Draw_Recover(1)
'显示图像
Call TURING.Pixel_Preview()

Sub 加载图片数据(图片路径)
    '加载图片
    Call TURING.Pixel_FromPicture(图片路径, 1)
    '备份图像
    Call TURING.Draw_Backups(1, 1)
End Sub
Function 屏幕区域找色(左, 上, 右, 下, 颜色值, 查找方向, 相似度)
    '恢复图像
    Call TURING.Draw_Recover(1, 1)
    '区域找色
    屏幕区域找色 = Split(TURING.FindColor(左, 上, 右, 下, _
            颜色值, 查找方向, 相似度), ",")
End Function
