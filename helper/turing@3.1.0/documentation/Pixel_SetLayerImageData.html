命令名称 Pixel_SetLayerImageData 配置图层图像数据
命令功能 对拆分的颜色图层设置为图像像素数据
语法格式 TURING.Pixel_SetLayerImageData(Index)
参数说明 Index：整数型，颜色图层数据索引号（顺序：蓝绿红）
返回值 无
按键精灵
复制代码 
'获取文件图像数据
Call TURING.Pixel_FromPicture("C:\504.bmp")
'色调分离处理
Call TURING.Filter_Posterization(4)
'颜色分层切割
n = TURING.Pixel_ColorImageData(3, "15")
For i = 0 To n
    '配置切割图像数据，进行预览（顺序：蓝绿红）
    Call TURING.Pixel_SetLayerImageData(i)
    Call TURING.Pixel_Preview()
Next
