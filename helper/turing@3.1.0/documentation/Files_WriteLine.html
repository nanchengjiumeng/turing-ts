命令名称 Files_WriteLine 写入一行文本内容
命令功能 在文件末尾写入一行文本内容
语法格式 TURING.Files_WriteLine(iText, SavePath)
参数说明 iText：字符串型，文本内容
SavePath：字符串型，保存文件路径（后缀名任意！）
返回值 无
按键精灵
复制代码 
Call TURING.Files_WriteLine("这是第1行", "C:\文本内容.txt")
Call TURING.Files_WriteLine("这是第2行", "C:\文本内容.txt")
Call TURING.Files_WriteLine("这是第3行", "C:\文本内容.txt")
Call TURING.Files_WriteLine("这是第4行", "C:\文本内容.txt")
Call TURING.Files_WriteLine("这是第5行", "C:\文本内容.txt")
