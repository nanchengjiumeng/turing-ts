描述说明 欢迎使用“虹鱼图灵识别”系统，首次使用插件说明
调用方法 创建对象的方式，脚本（每个线程）开头调用一次：
Set TURING = CreateObject("TURING.FISR")
注意事项 把插件放在“c:\TURING\”目录中，新建一个“注册.bat”文件，用记事本打开输入以下三行内容：
%1 mshta vbscript:CreateObject("Shell.Application").ShellExecute("cmd.exe","/c %~s0 ::","","runas",1)(window.close)&&exit
cd /d "%~dp0"
regsvr32 c:\TURING\TURING.dll
保存后，双击该文件来注册一下插件，【Win10系统】请右键管理员身份运行“注册.bat”文件。
这里必须使用绝对路径，以免有别人把“TURING.dll”释放在系统目录造成版本错误。
可以运行脚本输出版本号进行确认：
Set TURING = CreateObject("TURING.FISR")
TracePrint TURING.Version()

操作步骤 图灵识别的五大操作步骤：
【注意：图片要处理成“黑底白字”，才能进行切割识别】
1、来源：获取图像数据（有6种：获取屏幕像素，获取图片像素，获取内存像素，获取剪切板像素，获取鼠标（位置）图像像素，获取SVG数据文件像素）
2、滤镜：图像滤镜处理（有30种：二值化，色调分离，灰度，清除杂点，去掉直线，获取轮廓，提取色块，旋转矫正，倾斜纠正，颠倒颜色，膨胀腐蚀，抽骨细化，有效图像，等比缩放，缩放归一化，像素描边，颜色选留，固定旋转，筛选色点，固定镜像，延伸裁剪，祛除斑点，差异提取，色块选留，查找互补色，固定移位，固定柔化，固定扩散，电影底片，呈现暗调）
3、切割：图像字符切割（有5种：固定位置，随机方位，连通区域，范围投影，颜色分层）
4、字库：创建加载字库（有2种：生成自定义字库，创建系统字库）
5、识别：识别比对结果（有3种：标准和增强连体拆分识别，快速找字）
