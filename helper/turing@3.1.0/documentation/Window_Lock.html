命令名称 Window_Lock 窗口位置锁定
命令功能 锁定指定窗口句柄在屏幕上的位置
语法格式 TURING.Window_Lock(iHwnd, Flag)
参数说明 iHwnd：整数型，可选，窗口句柄(默认0:使用关联窗口句柄)
Flag：布尔型，可选，锁定状态(默认锁定True)
返回值 无
