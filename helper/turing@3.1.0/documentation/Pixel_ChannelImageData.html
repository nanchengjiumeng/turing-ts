命令名称 Pixel_ChannelImageData 获取通道图层像素
命令功能 对获取的图像像素数据进行RGB通道图层拆分（顺序：蓝绿红）
语法格式 TURING.Pixel_ChannelImageData(Model)
参数说明 Model：布尔型，可选，默认False全图，True单色图
返回值 无
按键精灵
复制代码 
'获取文件图像数据
Call TURING.Pixel_FromPicture("C:\f7d.jpg")
'通道图层
Call TURING.Pixel_ChannelImageData()
For i = 0 To 2
    '配置切割图像数据，进行预览（顺序：蓝绿红）
    Call TURING.Pixel_SetLayerImageData(i)
    Call TURING.Pixel_Preview()
Next
