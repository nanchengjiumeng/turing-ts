命令名称 Adb_KeyEvent 响应按键
命令功能 模拟键盘按键响应操作
语法格式 TURING.Adb_KeyEvent(Key)
参数说明 Key：整数型，键码
返回值 无
按键精灵
复制代码 
'关联调试桥（仅需脚本开头调用一次）
Call TURING.Adb_Link("C:\Program Files (x86)\dnplayer2\adb.exe")
'按Home键
Call TURING.Adb_KeyEvent(3)
'按音量+
Call TURING.Adb_KeyEvent(24)
