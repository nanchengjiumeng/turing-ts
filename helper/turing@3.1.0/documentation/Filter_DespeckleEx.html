命令名称 Filter_DespeckleEx 清除杂点_增强
命令功能 (针对黑白图像处理)清除杂点滤镜_增强
语法格式 TURING.Filter_DespeckleEx(Num, Through, Mode)
参数说明 Num：整数型，点数阈值
Through：布尔型，是否八向，默认True
Mode：整数型，颜色模式（默认：1去白点，0去黑点）
返回值 无
按键精灵
复制代码 
'获取屏幕图像数据
Call TURING.Pixel_FromScreen(0,0,200,100)
'二值化处理
Call TURING.Filter_Binaryzation("128-255")
'显示获取的图像
Call TURING.Pixel_Preview()
'去除杂点干扰_增强
Call TURING.Filter_DespeckleEx(3, True, 1)
'图像预览
Call TURING.Pixel_Preview()
滤镜效果
