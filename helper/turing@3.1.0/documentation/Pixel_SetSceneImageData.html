命令名称 Pixel_SetSceneImageData 配置图像数据环境
命令功能 配置图像像素数据应用于图色场景
语法格式 TURING.Pixel_SetSceneImageData(Mode)
参数说明 Mode：整数型，场景模式(默认1找图场景,2绘图图像>)
返回值 无
按键精灵
复制代码 
'获取屏幕图像数据
Call TURING.Pixel_FromScreen(0, 0, 500, 500)
'二值化图像处理
Call TURING.Filter_Binaryzation("128-255")
'显示获取的图像
Call TURING.Pixel_Preview()
'设置图像像素数据为找图场景
Call TURING.Pixel_SetSceneImageData()
'进行区域找图（内部图像数据）
坐标 = Split(TURING.FindImage(0, 0, 500, 500, "C:\小图.bmp", 1), ",")
x = 坐标(0)
y = 坐标(1)
TracePrint x  '输出为图像数据坐标系
TracePrint y
'进行区域找图
坐标 = Split(TURING.FindImage(0, 0, 1024, 768, "C:\小图.bmp", 1), ",")
x = 坐标(0)
y = 坐标(1)
TracePrint x  '输出为屏幕坐标系
TracePrint y
