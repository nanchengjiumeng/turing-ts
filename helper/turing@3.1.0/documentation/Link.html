命令名称 Link 关联窗口
命令功能 关联指定的窗口句柄
语法格式 TURING.Link(hwnd, mode)
参数说明 hwnd：整数型，窗口句柄(窗口不能最小化)
mode：字符串型，可选，关联模式。
　　默认：普通窗口后台"normal"，
　　普通前台窗口"window"，
　　3D游戏窗口后台"gdi"(切换主题为Aero再试)，
　　Aero主题缩略图后台"aero"，
　　后台键鼠模式"km"。
返回值 无
按键精灵
复制代码 
'Set TURING = CreateObject("TURING.FISR")
窗口句柄 = 1967868
'关联句柄
Call TURING.Link(窗口句柄)
'获取指定句柄的窗口大小
窗口大小 = Split(TURING.Window_GetSize(),",")
宽 = 窗口大小(2) - 窗口大小(0)
高 = 窗口大小(3) - 窗口大小(1)
'获取屏幕图像数据
Call TURING.Pixel_FromScreen(0, 0, 宽, 高)
'图像预览
Call TURING.Pixel_Preview()
'取消句柄的关联
Call TURING.UnLink()
