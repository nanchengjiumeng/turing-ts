命令名称 Incise_AddCharData 追加图像数据为切割数据
命令功能 追加图像数据为切割字符数据
语法格式 CharNum = TURING.Incise_AddCharData(Left, Top)
参数说明 Left：整数型，可选，设置左边坐标（默认0）
Top：整数型，可选，设置顶上坐标（默认0）
返回值 CharNum：整数型，字符切割数量
按键精灵
复制代码 
'获取屏幕图像数据
Call TURING.Pixel_FromScreen(0, 0, 100, 100)
'二值化处理
Call TURING.Filter_Binaryzation("168-255")
'显示图像
Call TURING.Pixel_Preview()
'固定位置切割字符
数量 = TURING.Incise_FixedLocation(0, 0, 50, 50, 1, 1)
'获取切割字符数据内容
数据 = Split(TURING.Incise_GetCharData(), "|")
切割 = Split(数据(0), ",")
TracePrint 切割(0) 
TracePrint 切割(1)
'清除所有字库
Call TURING.Incise_EraseData()
左 = 10: 上 = 10
'追加字库
数量 = TURING.Incise_AddCharData(左, 上)
TracePrint 数量
'获取切割字符数据内容
数据 = Split(TURING.Incise_GetCharData(), "|")
切割 = Split(数据(0), ",")
TracePrint 切割(0)  '被修改后的 左
TracePrint 切割(1)  '被修改后的 上
