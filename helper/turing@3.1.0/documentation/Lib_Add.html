命令名称 Lib_Add 添加新的识别字库
命令功能 添加新的识别字库数据到指定编号
语法格式 TURING.Lib_Add(Serial)
参数说明 Serial：整数型，可选，字库编号（默认1，上限20）
返回值 无
按键精灵
复制代码 
'加载识别字库
数量 = TURING.Lib_Load("C:\识别库1.lib")
'添加识别字库数据到指定编号
Call TURING.Lib_Add()   '默认1
'加载识别字库
数量 = TURING.Lib_Load("C:\识别库2.lib")
Call TURING.Lib_Add(2)
'切换使用指定编号的字库
Call TURING.Lib_Use()   '默认1
'切换使用指定编号的字库
Call TURING.Lib_Use(2)
