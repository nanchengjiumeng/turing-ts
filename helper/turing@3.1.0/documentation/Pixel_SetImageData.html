命令名称 Pixel_SetImageData 配置拆分图像数据
命令功能 对拆分的图像像素数据进行设置
语法格式 TURING.Pixel_SetImageData(Serial)
参数说明 Serial：整数型，指定数据编号（默认1，上限10个）
返回值 无
按键精灵
复制代码 
'获取屏幕图像数据
Call TURING.Pixel_FromScreen(50,50,250,150)
'显示获取的图像
Call TURING.Pixel_Preview()
'进行图像数据拆分1
Call TURING.Pixel_CutImageData(0,0,100,100,1)
'进行图像数据拆分2
Call TURING.Pixel_CutImageData(100,0,200,100,2)
'显示拆分的图像1
Call TURING.Pixel_SetImageData(1)
Call TURING.Pixel_Preview()
'显示拆分的图像2
Call TURING.Pixel_SetImageData(2)
Call TURING.Pixel_Preview()
