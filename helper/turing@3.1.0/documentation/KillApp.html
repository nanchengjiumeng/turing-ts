命令名称 KillApp 结束进程
命令功能 结束进程ID
语法格式 TURING.KillApp(PID)
参数说明 PID：整数型，进程ID
返回值 无
按键精灵
复制代码 
//启动线程
BeginThread DEMO(0)
BeginThread DEMO(1)
BeginThread DEMO(2)
//定义全局数组变量
Global xxxx(2)
Sub DEMO(i)
    //带参数执行，并返回进程ID
    xxxx(i) = TURING.ShellRun("D:\TURING.exe C:\图灵.tls")
End Sub
//演示过程
Delay 5000
TracePrint xxxx(0)
TracePrint xxxx(1)
TracePrint xxxx(2)
//结束进程ID
Call TURING.KillApp(xxxx(0))
Call TURING.KillApp(xxxx(1))
Call TURING.KillApp(xxxx(2))
