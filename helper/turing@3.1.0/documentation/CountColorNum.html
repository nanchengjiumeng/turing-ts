命令名称 CountColorNum 统计指定颜色点数量
命令功能 统计内部图像数据上的颜色点数量
语法格式 Num = TURING.CountColorNum(Color)
参数说明 Color：字符串型，指定统计的颜色串BBGGRR-BDGDRD（例："0000FF-000080|00FFFF"）
返回值 Num：整数型，颜色点数量
按键精灵
复制代码 
'获取屏幕图像数据
Call TURING.Pixel_FromScreen(0, 0, 100, 100)
'进行指定颜色数量统计
数量 = TURING.CountColorNum("0000FF")
'数量 = TURING.CountColorNum("FFFFFF-101010")
'数量 = TURING.CountColorNum("0000FF|FFFFFF-101010")
'数量 = TURING.CountColorNum("0000FF-202020|FFFFFF-101010")
