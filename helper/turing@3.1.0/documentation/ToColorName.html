命令名称 ToColorName 得到颜色值的名称
命令功能 得到指定的16进制颜色值的中文名称
语法格式 Name = TURING.ToColorName(sColor)
参数说明 sColor：字符串型，16进制颜色值（格式：BBGGRR）
返回值 Name：字符串型，16进制颜色值的中文名称（黑、灰、白、红、橙、黄、绿、青、蓝、紫）
按键精灵
复制代码 
'想知道是什么颜色？运行看看就知道了
TracePrint TURING.ToColorName("0000FF")
TracePrint TURING.ToColorName("0FFAFF")
TracePrint TURING.ToColorName("AAFFDD")
TracePrint TURING.ToColorName("FF00FF")
TracePrint TURING.ToColorName("00FFFF")
TracePrint TURING.ToColorName("FFFFFF")
TracePrint TURING.ToColorName("000012")
