命令名称 GetValidArea 获取有效区域范围
命令功能 获取内部图像的有效区域范围
语法格式 Area = TURING.GetValidArea()
参数说明 无
返回值 Area：字符串型，有效图像数据的范围（格式：左,上,右,下）
按键精灵
复制代码 
'获取屏幕图像数据
Call TURING.Pixel_FromScreen(0,0,200,100)
'二值化处理
Call TURING.Filter_Binaryzation("128-255")
'显示获取的图像
Call TURING.Pixel_Preview()
'获取有效区域范围
区域 = Split(TURING.GetValidArea(),",")
'绘制区域范围
Call TURING.Draw_Rectangle(区域(0), 区域(1), 区域(2), 区域(3))
'显示获取的图像
Call TURING.Pixel_Preview()
