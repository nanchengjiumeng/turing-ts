命令名称 SaveImageData 图像数据保存为图片
命令功能 将内部的图像数据保存为.bmp图片文件
语法格式 TURING.SaveImageData(ImagePath)
参数说明 ImagePath：字符串型，保存图片文件的路径
返回值 无
按键精灵
复制代码 
'获取屏幕像素数据
Call TURING.Pixel_FromScreen(0, 0, 800, 600)
'进行二值化滤镜处理
Call TURING.Filter_Binaryzation("0-178")
'将图像数据保存为图片
Call TURING.SaveImageData("C:\滤镜效果.bmp")
'显示像素图像数据
Call TURING.Pixel_Preview()
