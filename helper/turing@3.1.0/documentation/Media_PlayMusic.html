命令名称 Media_PlayMusic 播放音乐
命令功能 播放音乐（支持在线播放）
语法格式 TURING.Media_PlayMusic(FileName)
参数说明 FileName：字符串型，音乐文件路径（支持在线：http://*）
返回值 无
按键精灵
复制代码 
'本地音乐播放
声音文件 = "C:\TURING\图灵.mp3"
Call TURING.Media_TextToSpeech("欢迎使用图灵识别", 0, 声音文件)
Call TURING.Media_PlayMusic(声音文件)
Delay 2000    '播放时间
'在线音乐播放
Call TURING.Media_PlayMusic("http://www.turing.video/img/music.mp3")
Delay 150000    '播放时间

EndScript
'脚本结束后，记得要停止音乐播放，不然声音会卡进程!
Sub OnScriptExit()
    Call TURING.Media_StopMusic() 
End Sub
