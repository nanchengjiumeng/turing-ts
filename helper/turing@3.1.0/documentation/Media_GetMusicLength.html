命令名称 Media_GetMusicLength 获取播放时间长度
命令功能 获取播放时间长度(单位：毫秒)
语法格式 Length = TURING.Media_GetMusicLength()
参数说明 无
返回值 Length：整数型，音乐播放时间长度（单位：毫秒）
按键精灵
复制代码 
'在线音乐播放
Call TURING.Media_PlayMusic("http://www.turing.video/img/music.mp3")
'获取媒体的长度
播放时间 = TURING.Media_GetMusicLength()
TracePrint 播放时间 
Delay 播放时间

EndScript
'脚本结束后，记得要停止音乐播放，不然声音会卡进程!
Sub OnScriptExit()
    Call TURING.Media_StopMusic() 
End Sub
