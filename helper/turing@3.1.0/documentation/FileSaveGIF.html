命令名称 FileSaveGIF 生成GIF文件
命令功能 对多张图片合成GIF文件
语法格式 TURING.FileSaveGIF(LoadName, SaveName, Delay)
参数说明 LoadName：字符串型，加载多张图片（jpg,bmp,png），但必须是同尺寸文件（格式：图片1|图片2|…）
SaveName：字符串型，保存文件路径
Delay：整数型，可选，延迟时间（默认100毫秒）
返回值 无
按键精灵
复制代码 
Dim 文件名(5) 
文件名(0) = "C:\验证码\1.jpg"
文件名(1) = "C:\验证码\2.jpg"
文件名(2) = "C:\验证码\3.jpg"
文件名(3) = "C:\验证码\4.jpg"
文件名(4) = "C:\验证码\5.jpg"
文件名(5) = "C:\验证码\6.jpg"
'生成GIF动画图片
Call TURING.FileSaveGIF(Join(文件名,"|"), "C:\动画.gif")
