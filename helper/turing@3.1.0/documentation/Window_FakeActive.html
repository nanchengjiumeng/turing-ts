命令名称 Window_FakeActive 窗口假激活
命令功能 对窗口进行假激活后并获得焦点（同鼠标点击窗口后的效果，但不会置前）
语法格式 TURING.Window_FakeActive(iHwnd)
参数说明 iHwnd：整数型，可选，窗口句柄(默认0:使用关联窗口句柄)
返回值 无
按键精灵
复制代码
