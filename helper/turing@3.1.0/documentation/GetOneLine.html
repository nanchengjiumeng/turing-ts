命令名称 GetOneLine 指定端点之间的线段
命令功能 获取内部图像数据上的指定两端点之间的线条坐标
语法格式 Pos = TURING.GetOneLine(x1,y1,x2,y2)
参数说明 x1：整数型，起点X坐标
y1：整数型，起点Y坐标
x2：整数型，终点X坐标(最大分辨率为2000像素)
y2：整数型，终点Y坐标(最大分辨率为2000像素)
返回值 Pos：字符串型，线条上所有坐标（格式:x坐标,y坐标-x坐标,y坐标…）
按键精灵
复制代码 
'获取文件图像数据
Call TURING.Pixel_FromPicture("C:\验证码.BMP")
'二值化
Call TURING.Filter_Binaryzation("0-128")
'指定两个端点坐标
数据 = TURING.GetOneLine(23, 7, 69, 30)
端点 = Split(数据, "-")
数量 = UBound(端点)
For m = 0 To 数量
    TracePrint 端点(m)   '输出线条坐标数据
    坐标 = Split(端点(m), ",")
    x = 坐标(0)
    y = 坐标(1)
    'TracePrint x
    'TracePrint y
    '画颜色点
    Call TURING.Draw_Point(x, y)
Next
'显示图像
Call TURING.Pixel_Preview()
